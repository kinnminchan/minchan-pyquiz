# Minchan's Python Quiz
해당 문제들은 실제 고등학생 정보 교과 과외 수업에서 사용된 문제들입니다.

## 교과 외 내용 정리
문제를 제작하는 과정에서, 고등학교 정보 교과서에 포함되지 않은 일부 문법이 사용되었습니다. 🥲

1. 축약 연산자
    > 기존 연산을 간단히 표현할 수 있는 연산자입니다.
    > 
    > - `+=` 연산자
    > 
    >   > `a = a + b`를 `a += b`와 같이 표현할 수 있습니다.
    >   > 
    >   > ```python
    >   > total = 5
    >   > total += 3  # total = total + 3
    >   > print(total)  # 출력: 8
    >   > ```
    > 
    > - `*=` 연산자
    > 
    >   > `a = a * b`를 `a *= b`와 같이 표현할 수 있습니다.
    >   > 
    >   > ```python
    >   > total = 4
    >   > total *= 5  # total = total * 5
    >   > print(total)  # 출력: 20
    >   > ```
    > 
    >  `+=`, `-=`, `*=`, `/=`, `//=`, `%=` 등은 모두 **축약 연산자**라고 합니다.

2. `len()` 함수
    > 리스트, 문자열, 튜플 등 여러 개의 요소를 가지는 자료형의 **요소 개수(길이)**를 구하는 함수입니다.
    > 
    > ```python
    > arr = [5, 10, 15, 20]
    > print(len(arr))
    > ```
    > 
    > 위 코드에서 `arr`에는 4개의 요소가 있으므로,
    > `len(arr)`의 결과는 **4**입니다.
