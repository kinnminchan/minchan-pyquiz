## 17번 문제

정답: 6

해설:  
(1) 사용자가 1을 입력하면 i = 1
(2) while문 반복: i < 5
    i = 1 → arr[1] * (1 % 2) = 2 × 1 = 2 → total = 2
    i = 2 → arr[2] * (2 % 2) = 3 × 0 = 0 → total = 2
    i = 3 → arr[3] * (3 % 2) = 4 × 1 = 4 → total = 6
    i = 4 → arr[4] * (4 % 2) = 5 × 0 = 0 → total = 6
    i = 5 → 반복 종료
(3) 따라서 출력값은 6이다.

## 18번 문제

정답: 35

해설:  
(1) nums = [2, 4, 6, 8]
(2) 반복:
    01)
    i = 0 → j = 0 → total += 2 // 1 = 2 → total = 2

    02)
    i = 1 → j = 0 → 4 // 1 = 4 → total = 6
    j = 1 → 4 // 2 = 2 → total = 8

    03)
    i = 2 → j = 0 → 6 // 1 = 6 → total = 14
    j = 1 → 6 // 2 = 3 → total = 17
    j = 2 → 6 // 3 = 2 → total = 19

    04)
    i = 3 → j = 0 → 8 // 1 = 8 → total = 27
    j = 1 → 8 // 2 = 4 → total = 31
    j = 2 → 8 // 3 = 2 → total = 33
    j = 3 → 8 // 4 = 2 → total = 35
(3) 따라서 최종 total = 35이다.

## 19번 문제

정답: 6

해설:  
(1) 사용자가 1을 입력하면 i = 1
(2) while 반복문: i < 4
    01)
    i = 1 → (nums[1] + 1) // 5 = (10 + 1) // 5 = 11 // 5 = 2 → total = 2

    02)
    i = 3 → (nums[3] + 3) // 5 = (20 + 3) // 5 = 23 // 5 = 4 → total = 6
(3) 반복 종료 후 출력값은 6이다.
