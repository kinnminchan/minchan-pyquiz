## +4번 문제

정답: 22
해설:

(1) 조건은 값의 홀짝(nums[i] % 2) 과 인덱스의 홀짝(i % 2) 이 같은 경우이다.

    i = 0 → nums[0] = 4
    4 % 2 = 0, 0 % 2 = 0 → 조건 만족 → 추가
    
    i = 1 → nums[1] = 3
    3 % 2 = 1, 1 % 2 = 1 → 조건 만족 → 추가
    
    i = 2 → nums[2] = 6
    6 % 2 = 0, 2 % 2 = 0 → 조건 만족 → 추가
    
    i = 3 → nums[3] = 1
    1 % 2 = 1, 3 % 2 = 1 → 조건 만족 → 추가
    
    i = 4 → nums[4] = 8
    8 % 2 = 0, 4 % 2 = 0 → 조건 만족 → 추가

(2) result = [4, 3, 6, 1, 8]
(3) 합 = 4 + 3 + 6 + 1 + 8 = 22
(4) 따라서 출력값은 22이다.

## +10번 문제

정답: 20
